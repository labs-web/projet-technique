# Analyse Fonctionnelle - V5 : Interactivité Alpine.js

## 1. Introduction
Refonte de la couche interactive du Front-Office en utilisant Alpine.js pour une meilleure maintenabilité et des interactions plus riches (modales, transitions).

## 2. Acteurs
- **Visiteur** : Utilisateur utilisant l'interface publique.

## 3. Règles de Gestion (Business Rules)
### UX
- **Fluidité** : Utilisation de transitions douces pour l'apparition/disparition éléments.
- **Propreté** : Remplacement des scripts Vanilla "spaghetti" par des composants déclaratifs Alpine.js.

## 4. Fonctionnalités
### Visiteur
- **Interface réactive** (Utilisation de Modales pour détails ou confirmations, Dropdowns menus)
