# Standards de Documentation

## Format Markdown
- **PAS DE TABLEAUX** : L'utilisation de tableaux Markdown (`| col | col |`) est **STRICTEMENT INTERDITE**.
    - **Raison** : Difficulté de lecture et d'édition pour le développeur (problèmes d'alignement visuel).
    - **Alternative** : Utiliser des listes à puces ou des sections avec titres `###`.
- **Langue** : Français pour la documentation.
- **Forme** : Clair, concis, aéré.
